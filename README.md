## Deploy to Vercel

### Option A: Drag & Drop (Fastest)

1. Go to [vercel.com](https://vercel.com)
2. Click "Add New" → "Project"
3. Drag the `keystone-digital` folder onto the upload area
4. Done! Get auto-generated URL

### Option B: GitHub (Better for updates)

1. Create GitHub repo: `keystone-digital`
2. Push files to repo
3. Import repo in Vercel
4. Auto-deploy on every push

---

## Connect Domain

1. In Vercel dashboard → Project Settings → Domains
2. Add `keystonedigitalcorp.com`
3. Update DNS at your registrar:
   - Nameservers: `ns1.vercel.com`, `ns2.vercel.com`
   - OR A record: `76.76.21.21`
4. Wait 5-30 min for propagation
5. SSL auto-enabled
